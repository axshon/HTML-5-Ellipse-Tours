This is a demo of the HTML5 localStorage API (with a little jQuery UI thrown in for good measure).

Create some boxes, move them around, change their colors, then refresh the page to see that everything is remembered. Open this page in a second window and see that both windows stay in sync.

Written by Ian Gilman as part of a series of demos for the book HTML5 for .NET Developers, co-authored by Jim Jackson II; code available on Github.
